#  Analyse de performance en football — Données GPS & IMU  

**Celine BENDEKOUM — M1 2SIA**

##  Objectif  

Développer un outil de suivi de performance destiné à un club de football (entraîneur, préparateur physique, analyste) à partir de données **GPS/IMU**.  

L’objectif est de produire des indicateurs clairs, exploitables et rapides pour aider à la prise de décision.

---

##  Fonctionnalités  

- Nettoyage et prétraitement des données  
- Calcul de KPI (distance, vitesse, accélérations, charge…)  
- Analyse exploratoire des données  
- Visualisation des performances  
- Comparaison joueurs / matchs  

---

##  Technologies utilisées  

- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Jupyter Notebook
